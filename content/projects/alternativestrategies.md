Alternative Strategies connecte talents ambitieux et entreprises dynamiques dans la finance luxembourgeoise, avec un focus sur intégrité et excellence. J'ai développé leur site professionnel pour services recrutement et coaching.

## Le défi

**Problème initial** : Site WordPress lent (LCP 4s+), contenu difficile à gérer, visibilité SEO faible sur Luxembourg/Toulouse.

**Objectifs** :

- Vitesse <1s mobile
- CMS autonome pour offres/actualités
- Design premium reflétant valeurs excellence/transparence
- SEO local Luxembourg + France

## Ma solution technique

**Stack retenue** :

- **Nuxt 3** : Performances SSR + SEO natif
- **Strapi CMS** : Gestion offres emplois/articles
- **Tailwind CSS** : Design responsive pro

### Architecture

- Nuxt 3 (Nitro SSR)
- Strapi headless CMS
- Tailwind responsive with Nuxt UI

## Fonctionnalités développées

- Présentation services recrutement international finance
- Section coaching + ateliers Luxembourg sectoriel
- Strapi CMS dynamique (offres, articles autonomes)
- Design élégant valeurs entreprise
- Navigation intuitive tous appareils
- Optimisation SEO complète (schema.org, structured data)
